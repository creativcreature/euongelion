# App Store Asset Tracker

## Required visual assets

- App icon 1024x1024 (no alpha)
- iPhone screenshots
- iPad screenshots (if iPad supported)
- Optional app preview videos

## Metadata limits

- Description <= 4000 chars
- Keywords <= 100 chars

## Checklist state

- [ ] Icon validated
- [ ] Screenshot set complete
- [ ] Optional preview videos finalized
- [ ] Description validated
- [ ] Keywords validated
