# Methodology Traceability Matrix

## Method IDs

- M00 Unified Methodology
- M01 Site Architecture
- M02 SaaS Web UX Patterns
- M03 IA Research and Validation
- M04 Navigation Design
- M05 Accounts and Self-Service
- M06 Strategic IA + Conversion UX

## Mapping policy

1. Every feature PRD must reference at least one of M01-M06 plus M00.
2. Every methodology must map to at least one feature PRD.
3. No orphan methodology principle is allowed.

## Category defaults

- Governance: M00
- Homepage UX: M01, M02, M06
- Navigation: M01, M04, M06
- Typography/Motion: M02, M06
- Soul Audit/Devotional Engine: M02, M03, M06
- Devotional UX/Self-service: M04, M05, M06
- Data & privacy: M05
- Reliability/Performance/Accessibility: M00, M02
- Billing/iOS readiness: M00, M05
