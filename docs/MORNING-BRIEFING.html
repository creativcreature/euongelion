Key Recommendation

App vs Web App

Recommended Approach
Web-First PWA
Progressive Web App with "Add to Home Screen" — native apps later if justified by demand

Why: Maximum accessibility (no app store friction), faster time-to-market, single codebase, works offline, supports push notifications. Your audience includes people hesitant to "commit" — a URL is lower friction than an app download.

Full analysis with 7 decision points in /docs/decisions/APP-VS-WEB-APP.md

Important Finding

Accessibility Audit

Critical Issue Found
Gold on Scroll Fails Contrast
#C19A6B on #F7F3ED = 2.8:1 ratio (needs 4.5:1 for small text)

Solution provided: Use --gold-accessible: #8B6914 for text on light backgrounds. This darker gold achieves 5.1:1 contrast while maintaining brand feel.

Full audit with code fixes in /docs/research/ACCESSIBILITY-AUDIT.md
