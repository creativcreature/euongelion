# /drafts/ — Content Awaiting Review

This folder contains AI-generated devotionals that need founder review.

## Subfolders

| Folder                | Content Type                                                    |
| --------------------- | --------------------------------------------------------------- |
| `/apologetics/`       | Why Believe, Islam, Judaism, Spirituality, Historicity, Science |
| `/doctrine/`          | What We Believe, Hard Doctrines                                 |
| `/disciplines/`       | Way of Jesus, Learn to Pray, Fasting, Sabbath, Listening        |

| `/audience-branches/` | Grieving, Anger, Wilderness, etc.                               |

| `/story-of-god/`      | 52-week meta-narrative                                          |

## File Naming

```
[series-slug]-day-[##].md
```

Example: `welcome-to-euongelion-day-01.md`

## What Happens Next

1. Founder reviews content in this folder
2. If approved → moves to `/in-review/` or `/approved/`
3. If needs revision → notes added, stays here
4. Once finalized → moves to `/final/`

## Review Checklist

- [ ] Theological accuracy
- [ ] EUONGELION voice (honest, warm, intelligent)
- [ ] Citations verified
- [ ] Hebrew/Greek accurate
- [ ] Christ connection clear
- [ ] Interactive elements included
