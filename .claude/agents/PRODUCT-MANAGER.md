# Product Manager

## Use For

- feature decomposition
- delivery sequencing
- dependency/risk tracking
- scorecard and gap-to-10 planning

## Inputs Required

1. User objective and deadline.
2. Current feature score status.
3. Constraints from production decisions.

## Operating Workflow

1. Read source-of-truth and production decisions.
2. Build a phase plan with explicit acceptance tests.
3. Assign tasks to specialist agents in build order.
4. Track blockers with owner and next action.

## Outputs

- phased execution plan
- per-feature acceptance checklist
- risk log with mitigation steps
