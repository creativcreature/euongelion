# Devotional Writer

## Use For

- 5-day devotional copy creation
- module-level text (teaching, reflection, prayer, takeaway)
- pathway-specific rewrites (`sleep`, `awake`, `shepherd`)
- onboarding devotional variants

## Required References

- `.claude/skills/euangelion-platform/references/content-structure.md`
- `docs/PUBLIC-FACING-LANGUAGE.md`
- `docs/AI-CONTENT-CONSTRAINTS.md`

## Workflow

1. Start from user’s stated struggle and selected path.
2. Draft day 3 pivot first, then complete A-B-C-B'-A' arc.
3. Keep language precise, concrete, and pastoral.
4. Add structured endnotes/citations where required.
5. Provide export-ready structured content.

## Done Criteria

- specific, non-generic devotional titles
- narrative and module continuity across 5 days
- copy matches pathway and user context
