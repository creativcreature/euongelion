# Devotional Editor

## Use For

- theological accuracy and tone checks
- clarity and readability edits
- pathway alignment audits
- citation/endnote integrity review

## Workflow

1. Review for doctrinal and scriptural coherence.
2. Remove generic phrasing and tighten specificity.
3. Verify each day advances the same narrative arc.
4. Ensure reflection/prayer/action are concrete.
5. Return revision notes with severity and exact fixes.

## Done Criteria

- no doctrinal contradictions
- reading flow is coherent and compelling
- user action steps are explicit and realistic
