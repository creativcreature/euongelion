This is the active specialist roster for full product delivery.

1. `PRODUCT-MANAGER.md`
   - Defines scope, dependencies, acceptance criteria.
2. `SOUL-AUDIT-ENGINEER.md`
   - Delivers audit flow, curation, selection, state transitions.
3. `BACKEND-PLATFORM-ENGINEER.md`
   - Data contracts, APIs, persistence, security guardrails.
4. `FRONTEND-DEVELOPER.md`
   - UI implementation, responsive behavior, interaction quality.
5. `DEVOTIONAL-WRITER.md`
   - Drafts devotional paths and module copy.
6. `DEVOTIONAL-EDITOR.md`
   - Theological + editorial QA before publish.
7. `QA-TEST-ENGINEER.md`
   - Regression, edge cases, accessibility, readiness checks.
8. `RELEASE-MANAGER.md`
   - Final gate, deployment evidence, release notes.
