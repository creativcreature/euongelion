**Version:** 1.0  
**Last Updated:** January 16, 2026  
**Purpose:** Image treatment standards for all visual content

## PHILOSOPHY

**Photography serves the Word.**

Images exist to:

1. Create contemplative stillness
2. Evoke emotional resonance
3. Support theological themes
4. Never distract from text

**Reference:** Caravaggio's chiaroscuro - single-source lighting, subjects emerging from darkness

---

## THE CHALLENGE

**wokeGod doesn't control all images.**

Sources include:

- Substack-embedded photos (already present in HTML)
- User uploads (varying quality/style)
- External links (inconsistent)
- Stock photography (if used)

**Solution:** Treatment standards that make ANY image feel cohesive

## ACCEPTABLE IMAGE SOURCES

### Priority Order:

1. **Original Caravaggio-Style Photography**
   - Single-source lighting
   - Deep shadows
   - Subjects emerging from darkness
   - Timeless objects (bread, water, oil, thorns, linen, stone)

2. **Substack Embedded Images**
   - Already present in devotional HTML
   - Apply filters/treatments as needed
   - Crop to preferred aspect ratios

3. **User Uploads**
   - Accept with quality checks
   - Apply brand treatment automatically
   - Flag for review if badly off-brand

4. **Stock Photography (Curated)**
   - Unsplash: Religious, contemplative, nature
   - Pexels: Simple objects, textures, light
   - Must pass style filter

5. **Abstract/Texture Fallbacks**
   - When no suitable image available
   - Subtle paper texture, fabric, stone
   - Never generic stock backgrounds

## VISUAL STYLE CRITERIA

### ✅ PASS: On-Brand Images

**Mood:**

- Contemplative, quiet, reverent
- Dramatic but not theatrical
- Emotional weight, not sentimentality

**Lighting:**

- Single-source (window light, candle, spotlight)
- High contrast (deep shadows + bright highlights)
- Chiaroscuro effect
- Directional, intentional

**Subject Matter:**

- Biblical objects: bread, wine, oil lamps, scrolls, fabric
- Natural elements: water, stone, wood, earth, plants
- Hands (prayer, work, touch)
- Architectural elements (columns, doorways, stairs)
- Abstract textures (paper, cloth, stone grain)

**Composition:**

- Simple, uncluttered
- Clear focal point
- Generous negative space
- Intentional framing

**Color:**

- Muted, desaturated
- Earth tones preferred (brown, beige, gray, olive)
- Blacks and whites with midtones
- Accent colors subtle (burgundy, gold, blue - from brand palette)

**Tonality:**

- Rich blacks, deep shadows
- Controlled highlights (no blown-out whites)
- Midtone detail preserved
- Film-like grain acceptable

---

### ❌ REJECT: Off-Brand Images

**Avoid:**

- Modern clutter (phones, cars, logos, screens)
- Bright/cheerful stock photo aesthetics
- Generic corporate imagery
- Overly posed/staged
- Busy backgrounds
- Flat lighting (no drama)
- Oversaturated colors
- Obvious CGI/filters
- Trendy effects (lens flare, bokeh overload)
- Cliché religious imagery (hands in prayer with perfect lighting)

**Red Flags:**

- Multiple light sources (looks confusing)
- Harsh shadows (looks cheap)
- Too much color variation
- No clear focal point
- Feels like clip art
- Dated effects (90s Photoshop filters)

## IMAGE TREATMENT SPECIFICATIONS

### Standard Processing Pipeline

**For ALL images entering the system:**

#### 1. **Desaturation**

```
Saturation: -20% to -40%
Purpose: Mute modern colors, create timeless feel
Exception: Keep full saturation if already muted
```

#### 2. **Contrast Adjustment**

```
Contrast: +10% to +20%
Blacks: Deepen slightly
Highlights: Protect from clipping
Purpose: Create depth, drama
```

#### 3. **Color Grading (Optional)**

```
Apply subtle warm or cool tone:
- Warm: +5% yellow/amber in midtones
- Cool: +5% blue in shadows
Purpose: Cohesive color story
```

#### 4. **Vignette (Subtle)**

```
Darkness: 10-15%
Size: Large (feathered edges)
Purpose: Draw eye to center, create depth
```

#### 5. **Grain (Optional)**

```
Amount: 5-10%
Size: Fine
Purpose: Film-like quality, hide digital artifacts
```

#### 6. **Sharpening**

```
Amount: 30-50%
Radius: 1.0
Purpose: Crisp details without harshness
```

## ASPECT RATIOS & SIZING

### Preferred Ratios

| Usage                 | Ratio | Dimensions (min) |
| --------------------- | ----- | ---------------- |
| **Hero (full-bleed)** | 16:9  | 1920 x 1080px    |
| **Featured**          | 3:2   | 1500 x 1000px    |
| **Square**            | 1:1   | 1080 x 1080px    |
| **Portrait**          | 4:5   | 1080 x 1350px    |
| **Vertical Story**    | 9:16  | 1080 x 1920px    |

### Cropping Guidelines

**Full-Bleed Hero:**

- Crop to 16:9
- Ensure focal point in center third
- Allow for text overlay space (top or bottom third clear)

**Contained Images:**

- Prefer 3:2 or 1:1
- Center important elements
- Leave breathing room around subject

**Portrait/Vertical:**

- Use sparingly (mobile emphasis)
- Good for single objects, hands, doorways

## LAYOUT TREATMENTS

### 1. Full-Bleed Hero

**Purpose:** Maximum impact, set emotional tone

```
Width: 100vw (edge-to-edge)
Height: 80vh (desktop), 60vh (mobile)
Object-fit: cover
Position: Top of section or day
Text overlay: Optional (high contrast required)
```

**Best for:**

- Day intro images
- Series launch
- Major emotional beats

**Requirements:**

- Minimum 1920px width
- Strong horizontal composition
- Clear focal point
- Sufficient contrast for text overlay

---

### 2. Contained Image (Standard)

**Purpose:** Support text, not overpower

```
Width: 100% of content area
Max-width: 1200px
Margin: 60px auto (top/bottom)
Border-radius: 0 (sharp edges preferred)
Shadow: Subtle or none
```

**Best for:**

- In-text illustrations
- Breaking up long reading sections
- Supporting visual metaphors

---

### 3. Inset Image

**Purpose:** Small supporting visual

```
Width: 400-600px
Float: Left or right
Margin: 20px (opposite side + bottom)
Object-fit: cover
Aspect: 1:1 or 4:3
```

**Best for:**

- Object close-ups
- Detail shots
- Margin illustrations

---

### 4. Background Image (Text Overlay)

**Purpose:** Create immersive section

```
Width: 100vw
Height: 100vh
Object-fit: cover
Opacity: 0.3-0.5 (faded)
Blend-mode: Multiply or darken
Text: High contrast, centered
```

**Best for:**

- Pull quotes
- Section dividers
- Day number screens

**Requirements:**

- Image must be dark or darkened heavily
- Text must pass WCAG contrast (4.5:1 minimum)
- Focal point should not compete with text

---

### 5. Image Grid (Rare)

**Purpose:** Show multiple related images

```
Layout: 2-column or 3-column
Gap: 20px
Aspect: All same (1:1 or 3:2)
Max images: 6 per grid
```

**Best for:**

- Series of objects
- Before/after comparisons
- Visual timeline

## TEXT OVERLAY SPECIFICATIONS

### Readable Text on Images

**Requirements:**

- Text color: White or black (never mid-gray)
- Background: 40-60% opacity dark overlay
- Contrast ratio: Minimum 4.5:1
- Font size: Larger than usual (24px minimum)
- Font weight: Bold or black weight

**Safe Zones:**

- Keep text in center 60% of image
- Avoid busy/detailed areas
- Test on mobile (smaller screen = harder to read)

**Overlay Methods:**

1. **Full Dark Overlay**

```css
.image-overlay {
  background: rgba(0, 0, 0, 0.5);
  /* Text on top */
}
```

2. **Gradient Overlay**

```css
.image-overlay {
  background: linear-gradient(
    to bottom,
    rgba(0, 0, 0, 0) 0%,
    rgba(0, 0, 0, 0.7) 100%
  );
  /* Text at bottom */
}
```

3. **Box Behind Text**

```css
.text-box {
  background: rgba(247, 243, 237, 0.95);
  padding: 40px;
  max-width: 600px;
  /* Overlaid on image */
}
```

## EXTERNAL ASSET HANDLING

### Substack Embedded Images

**Current State:**

- Already in HTML as `<img>` tags
- Hosted on Substack CDN
- Various sizes/qualities

**Treatment Process:**

1. **Extract Image URL** from HTML
2. **Analyze Image:**
   - Check resolution (min 1200px width)
   - Check aspect ratio
   - Assess style fit
3. **Apply Filters:**
   - Desaturate (-30%)
   - Increase contrast (+15%)
   - Add subtle vignette
4. **Crop if Needed:**
   - Prefer 16:9 or 3:2
   - Center focal point
5. **Generate Responsive Sizes:**
   - 1920px (desktop)
   - 1200px (tablet)
   - 800px (mobile)
6. **Lazy Load:** Don't load until needed

---

### User Upload Workflow

**Acceptance Criteria:**

```
✅ ACCEPT:
- Resolution: 1200px+ width
- File size: Under 5MB
- Formats: JPG, PNG, WebP
- Aspect ratio: 16:9, 3:2, 4:3, 1:1, 4:5
- Quality: Sharp, in-focus
- Style: Passes visual criteria (see above)

❌ REJECT:
- Resolution: Under 800px width
- File size: Over 10MB
- Formats: GIF, BMP, TIFF
- Heavily compressed/pixelated
- Completely off-brand (modern clutter, etc.)
```

**Processing:**

1. **Auto-Accept** if passes all criteria
2. **Flag for Review** if borderline
3. **Reject with Feedback** if fails
   - "Image resolution too low (need 1200px+)"
   - "Modern elements visible (try timeless objects)"
   - "Lighting too flat (need dramatic contrast)"

**Fallback:**

- Offer curated stock alternatives
- Suggest abstract texture
- Allow text-only section

## FALLBACK SYSTEM

### When No Good Image Available

**Option 1: Abstract Texture**

```
Use subtle paper, fabric, or stone texture
Opacity: 10-20%
Purpose: Add visual interest without distraction
```

**Option 2: Solid Color Background**

```
Scroll White (#F7F3ED) or Tehom Black (#1A1612)
With generous white space
Typography becomes the focus
```

**Option 3: Typographic Treatment**

```
Large pull quote or Hebrew term
No image needed
Architectural type layout
```

**Never:**

- Generic stock placeholder (person icon, etc.)
- Low-res stretched image
- Completely unrelated stock photo

## ACCESSIBILITY

### Alt Text Requirements

**Structure:**

```
[Subject] [Action/State] [Context if relevant]
```

**Examples:**

Good:

- "Loaf of bread on wooden table"
- "Hands cupped around candle flame"
- "Stone archway with sunlight streaming through"

Bad:

- "Image" (too vague)
- "Bread" (too brief)
- "A beautiful photograph of artisanal sourdough bread with rustic aesthetic on reclaimed wood" (too verbose)

**Guidelines:**

- 125 characters or less
- Describe what's visible, not interpretation
- No "image of" or "photo of" prefix
- Include relevant details for context

## OPTIMIZATION

### File Size Targets

| Usage             | Max Size (JPG) | Max Size (WebP) |
| ----------------- | -------------- | --------------- |
| Hero (1920px)     | 200KB          | 100KB           |
| Standard (1200px) | 120KB          | 60KB            |
| Thumbnail (400px) | 40KB           | 20KB            |

### Compression Settings

**JPG:**

- Quality: 75-85%
- Progressive scan
- Strip EXIF data

**WebP:**

- Quality: 80%
- Fallback to JPG for Safari

**PNG:**

- Only for transparency needs
- Otherwise convert to JPG/WebP
