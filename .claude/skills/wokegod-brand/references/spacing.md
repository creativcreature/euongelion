# wokeGod Spacing System

**Version:** 1.0

## PHILOSOPHY

**Generous white space = Sabbath rest = theological breathing room**

## BASE UNIT

8px grid system (everything divisible by 8)

## SCALE

- xs: 8px (tight)
- sm: 16px (close)
- md: 24px (comfortable)
- lg: 40px (generous)
- xl: 60px (expansive)
- 2xl: 80px (dramatic)
- 3xl: 120px (full breath)

## SECTION PADDING

- Desktop: 80-120px top/bottom
- Tablet: 60px top/bottom
- Mobile: 40px top/bottom
- Side margins: 60px (desktop), 24px (mobile)

## CONTENT SPACING

- Paragraph spacing: 1.5em (30px at 20px font)
- Section breaks: 80px minimum
- Image margins: 60px top/bottom
- Pull quote isolation: 120px top/bottom

## MAX WIDTHS

- Reading: 680px
- Display: 1200px
- Full-bleed: 100vw

## VERTICAL RHYTHM

Line-height drives vertical spacing:

- Body: 1.7 (34px at 20px) → use 32px or 40px spacing
- Headings: 1.1-1.3 → tighter spacing above/below

[See typography.md for line-height specs]
